A) Installation - top

Copy all files to your webserver.

Include JS files:

<script type='text/javascript' src='https://www.google.com/jsapi'></script>
<script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false"></script>
<script type="text/javascript" src="js/jquery.js"></script>
<script type="text/javascript" src="js/ultimate_flash_map_killer.js"></script>

Include CSS files:

<link href="css/ultimate_flash_map_killer.css" rel="stylesheet" type="text/css"/>
<link href="css/demo.css" rel="stylesheet" type="text/css"/>

Create a container for your map with search toolbar:

<div class="divToolbar">
<div class="searchmap"> Search Map: <input onkeypress="findMapAddress(event)" type="text" id="txtMapSearch" /></div>
</div>
<div id="map_canvas"></div>

Add Javascript code:

<script type="text/javascript">
var map; //Global Map object to perform multiple actions
$(document).ready(function() {

var poly=new Polygon();
var mapOptions = {
zoom: 2,
center: new google.maps.LatLng(24.886436490787712,80.2685546875),
mapTypeId:"roadmap"
};
map = new google.maps.Map(document.getElementById("map_canvas"), mapOptions);
var landmark= new LandMark(map);
landmark.loadLandMarks();
poly.createPolygon(map, {
country: ["Iran (Islamic Republic of)","Spain", "Mexico", "Greenland"],
strokeColor:"#A52A2A",
strokeOpacity:0.1,
strokeWeight:2,
fillColor:"#A52A2A",
fillOpacity:0.8,
hoverFillColor:"#FF0000",
hoverFillOpacity:0.1,
hoverStrokeColor:"#000",
hoverStrokeWeight:2,
hoverStrokeOpacity:0.5,
toolTip:true,
toolTipText:"Enter Tooltip Here",
onClickType:2,
onClickValue:{url:"http://www.google.com",popup:"<h3>HTML CODE HERE</h3>",callback:function(){alert('test')}}


});
function findMapAddress(e) {
if (e.keyCode==13)
{var geocoder = new google.maps.Geocoder();
var address = document.getElementById("txtMapSearch").value;

geocoder.geocode( { 'address': address}, function(results, status) {
if (status == google.maps.GeocoderStatus.OK) {
map.setCenter(results[0].geometry.location);
map.setZoom(11);

} else {
alert("Location not found!");
}
});
}
}

</script>

and you are done!!

C) JavaScript - top

This map imports the following Javascript files.

  1. jQuery
  2. Google Api Loader and Google Maps API
  3. ultimate_flash_map_killer.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. The base map is served by Google Maps API so you can extend the map to any length the API supports. Options configured here are

     

    var mapOptions = {
        zoom: 2,
        center:  new google.maps.LatLng(24.886436490787712,80.2685546875),
        mapTypeId:"roadmap"
    };

  3. The custom script is kept inside ultimate_flash_map_killer.js, you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method and the options available.

     

    // Activate Landmarks
    var landmark= new LandMark(map);
    landmark.loadLandMarks();

    // Loading Country or Region Overlays
    poly.createPolygon(map, {
    country: ["Iran (Islamic Republic of)","Spain", "Mexico", "Greenland"],

    Note the above set of parameters apply to a single polygon object or an array of objects. These polygons are loaded in this line:
    country: ["Saudi Arabia","Spain", "Mexico", "Greenland"],

    You can use any one of the following three options at one time with the above settings:

    1. country: ["Saudi Arabia","Spain", "Mexico", "Greenland"],
    2. continent: [poly.Australia]
    3. state: ["Texas","California","Florida", "North Dakota", "Alaska"],
    4. Country and States are supposed to be in quotes. Continents should be entered without quotes. List of supported countries, US states and continents is given ahead in this document. Names are case sensitive.

    // Configuring layer fill colors, borders or stroke colors, transparency. Colors are hex values supported by html. Opacity is between 0 and 1.
    // Configure similar settings for hover or mouse over effect
    strokeColor:"#A52A2A",
    strokeOpacity:0.1,
    strokeWeight:2,
    fillColor:"#A52A2A",
    fillOpacity:0.8,
    hoverFillColor:"#FF0000",
    hoverFillOpacity:0.1,
    hoverStrokeColor:"#000",
    hoverStrokeWeight:2,
    hoverStrokeOpacity:0.5,

    //Enable or Disable tooltip
    toolTip:true,
    toolTipText:"Australia",
    //Configure onClick functionality
    onClickType:2,
    onClickValue:{url:"http://www.google.com",popup:"<h3>HTML CODE HERE</h3>",callback:function(){alert('test')}}

    onClickType takes either 1 or 2 or 3 as value, based on which the functionality is selected from onClickValue
    1:         Go to URL defined in onClickValue:{url:"http://www.google.com"
    2:         Pop-Up box containing text defined in popup:"<h3>HTML CODE HERE</h3>",
    3:         Javascript callback as in callback:function(){       

D) Landmarks and XML - top

The map supports dynamic landmarks loaded from Landmarks.xml file. The structure is as follows:

<LandMark Lat="-33.873651" Lng="151.20688960000007" LandMarkType="TIME" Description="Sydney" Address="1 George St, Sydney NSW 2000, Australia" Offset="10" ImageUrl="images/redpin.png"/>

Specify the latitude and longitude position of the landmark. This is essential.
Next set the LandMarkType as either “ICON” to display an icon image defined in ImageUrl or “TIME” to display timezone with GMT offset defined in Offset.
